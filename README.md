# Inclusive AI Meeting Bot

## To run:

Set keys in .env (your actual API Keys)
    - an example format is shown in .env.example 

Open 2 terminals (RUN IN EXACT ORDER):
     1. cd backend --> `npm install` (once) --> `npm start`
     2. cd frontend --> `npm install` (once) --> `npm run dev`

Start a Zoom meeting.

Open the localhost Link provided by the frontend terminal (vite)
    - Paste Zoom Link into provided textbox.
